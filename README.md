# oculus_driver
ROS device driver for Blueprint Subsea's Oculus multibeam sonars


# Usage

```sh
roslaunch sonar_oculus sonar_oculus.launch

# play bag file
roslaunch sonar_oculus sonar_oculus.launch sonar:=false
```
